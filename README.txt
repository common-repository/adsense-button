Add a button for inserting Adsense code in your content

== Description ==

With this plugin, you can add different adsense code. When you are in the content editor, you will have a $ button to insert the ad. The ad can be align on the right/left/center like a picture.

== Installation ==

1. Upload `adsensebutton` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your adsense buttons in the settings
4. Click on the $ button in your content editor.
